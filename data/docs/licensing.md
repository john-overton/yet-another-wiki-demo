## License Management âš™ï¸

### Verifying License

1. Navigate to Settings > Licensing
2. Enter:
   - Email address
   - License key
3. Click "Verify License"

### License Format

```
XXXXX-XXXXXXXXXXXXXXXXXXX
```

:::info
License verification requires internet connection for initial validation.
:::
